🏋️‍♂️ Gym Landing Page

A modern and fully responsive Landing Page built with HTML, CSS, and JavaScript for a gym and fitness center.
The project features a friendly user interface, optimized for all devices, and highlights essential sections such as schedules, training classes, and contact details.

🚀 Features

📱 Fully Responsive: Works seamlessly on desktop, tablet, and mobile devices.

🎨 Modern & Friendly UI: Clean, engaging, and fitness-focused design.

🏋️‍♀️ Dedicated Sections: Hero, Working Time, About, Training Classes, Contact, and Footer.

⚡ Interactive Elements powered by JavaScript.

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

📂 Project Structure

project-folder/

│── index.html   # Main HTML file
│── gym.css    # Styles for the UI
│── gym.js    # JavaScript logic for interactivity
│── photos/      # Project images

⚙️ How It Works

Open the landing page in your browser.

Navigate through the sections:

Hero Section: Welcoming banner for the gym.

Working Time Section: Display of gym opening hours.

About Section: Information about the gym’s philosophy and mission.

Training Classes Section: Showcase of available fitness classes.

Contact Section: Contact form and location details.

Footer Section: Quick links and social media.

Enjoy a responsive and modern experience across all devices.

📦 Installation & Usage

Clone the repository:

git clone [https://github.com/hamou592/gym-landing-page.git](https://github.com/hamou592/Gym-website.git)


Open index.html in your browser.

Explore the responsive landing page.
